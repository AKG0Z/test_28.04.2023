Comparatif systèmes de gestion de base de données:


Il existe une multitude de solutions de gestion de bases de données. Chacune d’elles se prédestine à un usage, et y bonne à sa manière, dans ce qui suit un exposé des systèmes de gestion de base de données les plus utilisés, ainsi qu’une comparaison non exhaustive entre ces diverses applications.

Criteres d'evaluation :

Avant entrer en la matiere, il est a preciser que le present comparatif n'est pas a generaliser ou a prendre en absolut, en effet les appriciations varient selon le contexte d'utilisation et les criteres d'evaluation, nous concernant les criteres sont :

  • Le type de licence.
  • Langage de programmation.
  • Ressouces necessaires pour le bon fonctionnement.
  • Disponibilité des ressources et documentation.
  • Accessibilité et prise en main.
  • Les avantages.
  • Inconvénients.

Systemes de gestion de bases de données :

Pour les systeme de gestion de bases de données evalués, il sera question de :

  1. ORACLE
  2. SQLServer
  3. PostgreSQL
  4. MySQL
  5. mongoDB
  6. Cassandra

Le present lien renvoie à un tableau récapétulatif au pied de cette page.

1.ORACLE

  • Le type de licence : Commercial.
  • Langage de programmation : Java, C et C++.
  • Ressouces : Consommation de ressources élevée.
  • Disponibilité des ressources et documentation : Oracle Documentation Oracle Data base Insider.
  • Accessibilité et prise en main : Difficile à maîtriser.
  • Les avantages : Portabilité Régulièrement mis à jour. Solide support technique et documentation. Grande capacité.
  • Inconvénients : Cout des licences. Courbe d'apprentissage difficile.
  • Telecharger : ORACLE

2.SQLServer

  • Le type de licence : Licence propriétaire et EULA.
  • Langage de programmation : C++, C et C#.
  • Ressouces : Utilisation scalable des ressources.
  • Disponibilité des ressources et documentation : Documentation de SQL Server Microsoft SQL Server Blog.
  • Accessibilité et prise en main : Difficulté moyenne.
  • Les avantages : Support de base de données en Cloud.
  • Inconvénients : Cout des licences. Courbe d'apprentissage difficile.
  • Telecharger : SQLServer

3.PostgreSQL

  • Le type de licence : Licence PostgreSQL.
  • Langage de programmation : C.
  • Ressouces : Léger sur les ressources.
  • Disponibilité des ressources et documentation : Documentation : PostgreSQL Documentation Planet PostgreSQL.
  • Accessibilité et prise en main : Facile à utiliser.
  • Les avantages : open-source, prise en charge des types de données personnalisés, outils tiers faciles à intégrer, open-source et communautaire, personnalisable.
  • Inconvénients : Documentation incohérente. Manque d'instruments de reporting et d'audit.
  • Telecharger : PostgreSQL

4.MySQL

  • Le type de licence : Licence propriétaire Licence publique générale GNU version 2.
  • Langage de programmation : C et C++.
  • Ressouces : Poids léger sur les ressources.
  • Disponibilité des ressources et documentation : Documentation : MySQL Documentation Blog : The Oracle MySQL Blog
  • Accessibilité et prise en main : Facile à utiliser.
  • Les avantages : Open-source Large communauté, sécuritaire performant Compatible avec le cloud.
  • Inconvénients : Difficilement scalable Perte de performance avec de grand volumes de donnée.
  • Telecharger : MySQL

5.mongoDB

  • Le type de licence : Server Side Public License.
  • Langage de programmation : C++, Go, JavaScript et Python.
  • Ressouces : Consommation de mémoire elevé.
  • Disponibilité des ressources et documentation : MongoDB Documentation MongoDB Blog.
  • Accessibilité et prise en main : Facile à utiliser.
  • Les avantages : De très bonnes performances Facile à installer Base de données évolutive horizontalement, compatibilité facile avec d'autres modèles de données, accès simple aux données, stockage, saisie et récupération.
  • Inconvénients : Fonctionnalités limitées Utilisation de ressources non optimisée, insécurité des données, processus compliqué à interpréter dans d'autres langages de requête.
  • Telecharger : mongoDB

6.Cassandra

  • Le type de licence : Licence Apache 2.
  • Langage de programmation : Java.
  • Ressouces : Nécessite des ressources importantes.
  • Disponibilité des ressources et documentation : Apache Cassandra Documentation Cassandra Blog.
  • Accessibilité et prise en main : Syntaxe simple.
  • Les avantages : Évolutivité élevée, sécurité des données, lecture lente.
  • Inconvénients : Fonctionnalités limitées.
  • Telecharger : Cassandra
